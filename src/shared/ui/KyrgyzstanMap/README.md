# KyrgyzstanMap Component

Переиспользуемый компонент для отображения карты Кыргызстана с локациями.

## Описание

Компонент отображает стилизованную карту Кыргызстана с возможностью добавления локаций в виде красных сердечек. Карта включает в себя:
- Контур страны с региональными границами
- Декоративные иконки (горы, фабрика, дерево, поле, звезда)
- Интерактивные маркеры локаций
- Адаптивный дизайн

## Использование

### Базовое использование

```tsx
import KyrgyzstanMap from '@/shared/ui/KyrgyzstanMap'

function MyComponent() {
    return <KyrgyzstanMap />
}
```

### С кастомными данными

```tsx
import KyrgyzstanMap from '@/shared/ui/KyrgyzstanMap'

function MyComponent() {
    const locations = [
        { id: 1, x: 100, y: 80, name: "Бишкек" },
        { id: 2, x: 150, y: 120, name: "Ош" },
        { id: 3, x: 200, y: 60, name: "Нарын" }
    ]

    return (
        <KyrgyzstanMap 
            title="Наши офисы"
            subtitle="Мы работаем во всех регионах Кыргызстана"
            locations={locations}
        />
    )
}
```

## Props

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| `title` | `string` | `"Наши локации"` | Заголовок компонента |
| `subtitle` | `string` | `"Наша деятельность охватывает все 20 регионов Кыргызстана"` | Подзаголовок |
| `locations` | `Location[]` | `[]` | Массив локаций для отображения |
| `className` | `string` | `""` | Дополнительные CSS классы |

## Типы

### Location

```tsx
interface Location {
    id: number        // Уникальный идентификатор
    x: number         // Позиция по X (в координатах SVG)
    y: number         // Позиция по Y (в координатах SVG)
    name?: string     // Название локации (опционально)
}
```

## Особенности

- **Адаптивность**: Компонент автоматически адаптируется под размер экрана
- **Интерактивность**: Маркеры локаций имеют hover-эффекты
- **Кастомизация**: Легко настраивается через props
- **Доступность**: Поддерживает семантическую разметку

## Стилизация

Компонент использует CSS Modules. Основные классы:

- `.kyrgyzstanMap` - основной контейнер
- `.header` - блок с заголовком
- `.mapContainer` - контейнер карты
- `.locationMarker` - маркер локации
- `.heartIcon` - иконка сердца
